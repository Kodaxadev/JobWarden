Install on iPhone or iPad

  1. In Safari, tap the Share button.
  2. Choose Add to Home Screen.
  3. Tap Add — JobWarden opens like a normal app and works offline.

Install from the browser menu

  1. Open the browser menu (⋮ or ···, usually top-right).
  2. Choose Install app or Add to Home screen.

Private by design

Everything logged stays on this device — no account, no server, fully offline. Back up from inside the app (Export → Save full backup).

Not legal advice. No audio recording (illegal in California without all-party consent).
